# GNN-Assisted QAOA for the Traveling Salesman Problem

Hi! This is **Jalaj, Vibha, and Aarushi**.

This project explores a hybrid approach that combines **Graph Neural Networks (GNNs)** with the **Quantum Approximate Optimization Algorithm (QAOA)** to solve the **Traveling Salesman Problem (TSP)**.

## Project Overview

The Traveling Salesman Problem is a classic **NP-hard optimization problem** where the objective is to find the shortest possible route that visits each city exactly once and returns to the starting city.

Quantum algorithms, such as QAOA, are designed to solve combinatorial optimization problems. However, QAOA performance often depends strongly on how the algorithm is initialized. Poor initialization can lead to slow convergence or suboptimal solutions.

In this project, we investigate whether a **Graph Neural Network can learn useful structural patterns from TSP graphs** and use that knowledge to provide **better initial parameters for QAOA**.

This creates a **hybrid classical–quantum pipeline** where machine learning helps guide a quantum optimization algorithm.
